# api-shoeCatalogue

### Introduction

- It is the app that has the code that bridges two software programmes.
- It combines front end and back end.
- Use ajax calls

### API shoe catalogue allows users to:

- Filter by size and brand.
- Buy any shoe

### Admin can:

- Add shoes and update the existing shoe
